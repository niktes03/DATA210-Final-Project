# DATA210-Final-Project

## Overview
This project investigates whether features derived from a movie's plot overview, specifically sentiment polarity, alongside engagement metrics and release era can predict a film's TMDB vote average. Three models are compared: OLS Linear Regression, Bayesian Ridge Regression, and Gradient Boosting.

## How to Run

1. Clone the repository
2. Set up a virtual environment:
```bash
   python -m venv venv
   source venv/bin/activate
```
3. Install dependencies:
```bash
   pip install -r requirements.txt
```
4. Open `code.ipynb` in Jupyter and run all cells top to bottom
